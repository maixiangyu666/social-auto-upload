本项目基于https://github.com/dreammis/social-auto-upload.git  后期项目完整会合并到此项目 此项目作为临时GIT管理 

【项目协作公告】Social Auto Upload 项目任务认领启动！
大家好！为了让咱们的 Social Auto Upload 项目更好地协同开发，现正式启用任务协作表，欢迎大家认领任务，贡献力量！
核心协作流程
任务认领：
地址：[金山文档 | WPS 云文档] 任务名称 - 任务协作表.dbt (https://www.kdocs.cn/l/cmSuWfUyCMHi)
操作：在 “任务协作表” 中，选择 “状态” 为 “待认领” 的任务，填写你的名字和预估完成时间，即可认领。
任务执行：
请在预估时间内完成任务。
代码提交：请拉取项目后提交 PR 到主仓库 (https://github.com/maixiangyu666/social-auto-upload.git)。若没有权限，请联系我添加。
任务归还：
若认领后发现没时间或遇到困难，请及时将任务状态改回 “待认领”，放回任务池，方便他人接手。
若超过预估完成时间未完成，任务也将被视为自动归还。
新想法 / 迭代：
如有新功能想法或改进建议，也可记录在协作表中，大家讨论后细化为可执行任务。
重要提醒
爱心发电，灵活参与：大家根据自己的时间和兴趣参与，无需有压力。
避免重复劳动：认领前请务必查看任务状态，确保没有其他人正在处理。
保持沟通：遇到问题或需要帮助，欢迎在群里讨论。
